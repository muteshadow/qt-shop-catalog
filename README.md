# ShopCatalog - Qt Desktop Application

This is a desktop application built with Qt (C++) for managing a product catalog.
It allows users to browse products, view details, and manage a shopping cart and favorites list.

  <img src="docs/screenshots/main.png" width="480" alt="Головне вікно програми">
</p>

## ✨ Features

- Product catalog browsing
- Product details view
- Add/remove items to cart
- Favorites system
- Admin catalog editing
- SQLite database integration
- Dynamic UI generation (Qt Widgets)
